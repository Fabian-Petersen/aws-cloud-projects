## Key Metrics for Asset Work History to Consider

### Reliability & Availability

- Mean Time Between Failures (MTBF) — average operating time between breakdowns
- Mean Time To Repair (MTTR) — average time to restore an asset after failure
- Asset Availability % — (Uptime / Total Time) × 100
- Failure Rate — number of failures per period

### Maintenance Activity

- Work Order Count — total WOs raised per asset (breakdown by type: corrective, preventive, predictive)
- Planned vs Unplanned Maintenance Ratio — higher planned % indicates a proactive program
- PM Compliance % — percentage of scheduled PMs completed on time
- Work Order Backlog — open/overdue WOs per asset

### Cost

- Total Maintenance Cost — labor + parts + contractor costs per asset over time
- Cost per Work Order — average spend per job
- Maintenance Cost as % of Asset Replacement Value (RAV) — industry benchmark: healthy assets sit below 3%
- Parts Spend Trend — escalating parts costs can signal end-of-life

### Time & Efficiency

- Response Time — time from fault reported to work started
- Wrench Time % — actual time spent working vs waiting/traveling
- First-Time Fix Rate — WOs closed without a repeat visit

### Health & Condition

- Failure Mode Distribution — which failure modes recur most (useful for RCA)
- Condition Score Trend — if you capture inspection scores over time
- Remaining Useful Life (RUL) estimate — based on age, usage hours, or condition

### Usage-Based

- Operating Hours / Cycles since last service
- Hours between services — drift from scheduled intervals
- Odometer / Runtime at each failure event
